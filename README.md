# X-Raid
### Discord Advanced Raid Tool

Универсальный рейдтул/raidtool для Discord

Инструкция использования:
- Скачайте Python (желательно версии 3.13/3.14) на сайте https://www.python.org/downloads (нужно поставить галочку при установке: "Add python.exe to PATH")
- Добавьте токены аккаунтов в файл tokens.txt
- (Опционально) Добавьте прокси в файл proxies.txt (Их можно бесплатно брать с сайта https://proxyscrape.com/free-proxy-list. Главное надо выбирать протокол HTTP и задержку около 1000 мс)
- Запустите CLICK.bat
